# NeoFM

A high-performance, modern terminal file manager built with Go and Rust. NeoFM provides a sleek, intuitive TUI (Terminal User Interface) for efficient file system navigation and management across multiple platforms.

## Key Features
- **Cross-Platform:** Native support for both Windows and Linux.
- **Tabbed Interface:** Manage multiple directories simultaneously.
- **Built-in Editor:** Quick file editing with syntax highlighting.
- **Advanced Tools:** Integrated support for ZIP/Unzip and AES-256 encryption/decryption.
- **Permission Management:** Interactive file permission (chmod) viewer and editor.
- **Professional TUI:** Beautifully styled interface using `lipgloss` and `bubbletea`.

## Keyboard Shortcuts
- `F1`: Open Help Menu
- `Space`: Select multiple files
- `F10`: Access Tools Menu (Operations, Clipboard, Tools, System)
- `Tab`: Cycle through active tabs
- `Ctrl+N`: Create new file/folder
- `Ctrl+W`: Close current tab

## Installation
Pre-compiled binaries are available in the root directory:
- `NeoFM-bin.exe` (Windows)
- `NeoFM-bin` (Linux)
